# pyPlayMusic

play notes
play wave, wav, mp3... // in construction

with ( async || sync ) && note

## Quick Start
```python
from pyPlayMusic import playNotes

playNotes(codes=[57]).play() # play notes with async
time.sleep(0.3)
playNotes(codes=[57,60]).play()
time.sleep(0.3)
playNotes(codes=[57,60,63]).play()
time.sleep(0.3)

```
